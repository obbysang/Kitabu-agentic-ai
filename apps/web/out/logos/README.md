# Logos

## File Naming Conventions
- Base pattern: `logo-{variant}.{format}`
- Variants: `full-color`, `monochrome`, `inverted`
- Formats: `svg`, `png`, `jpg`
- Minimum dimensions: 300x300 for raster formats

## Usage Guidelines
- Use `logo-full-color` for standard brand surfaces
- Use `logo-monochrome` on low-contrast or print contexts
- Use `logo-inverted` on dark backgrounds
- Prefer `svg` in web apps for scalability; fall back to `png`/`jpg` when required
