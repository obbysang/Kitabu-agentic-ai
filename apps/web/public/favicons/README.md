# Favicons

## File Naming Conventions
- Standard ICO: `favicon.ico` with 16x16, 32x32, 48x48 sizes
- Additional formats: `favicon.svg`, `favicon-32x32.png`, `favicon-16x16.png`
- Apple touch icon: `apple-touch-icon.png` (180x180)
- Android Chrome: `android-chrome-192x192.png`, `android-chrome-512x512.png`

## Usage Guidelines
- Link `favicon.ico` and `favicon.svg` in HTML `<head>`
- Link Apple touch icon for iOS home screen
- Provide Android Chrome icons for PWA installs
- Keep all favicon assets under `favicons/` for consistent paths
